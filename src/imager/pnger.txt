pnger is a consoler application for viewing png files.

-------------
pnger FILE
-------------

It takes a single command line argument specifying which png file to view.

The following key bindings are defined.

[horizontal]
q:: quit the application
h:: scroll left 10% of the view
j:: scroll down 10% of the view
k:: scroll up 10% of the view
l:: scroll right 10% of the view
i:: zoom in by a factor of 2
o:: zoom out by a factor of 2
