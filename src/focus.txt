Focus {VERSION}
===============
Richard Uhler <ruhler@member.fsf.org>

Focus is a project which aims to produce a computing environment which allows
you to focus on the work you are doing. focus consists of the following
components.

[horizontal]
consoler:: Library for interacting with the user via keys and display.
fonter:: Library for drawing fonts on consoler displays.
boxer:: A box game.
filler:: A sample program to fill a display with color.
green:: A screen-like application window manager.
pdfer:: A pdf viewer.
pnger:: A png viewer.
sdlcsr:: A consoler server implemented on libSDL.
termer:: A terminal emulator.

Each of these components is described in more detail in the following
sections.
