NAME
----
cgreen - Launch a consoler client connecting to green server

SYNOPSIS
--------
cgreen [-e] client [client args ..]

DESCRIPTION
-----------
include::cgreen.txt[]

OPTIONS
-------
*--help*::
    Output a help message and exit.
*--version*::
    Output version information and exit.
*-e*::
    Wait for the client to exit before exiting.
